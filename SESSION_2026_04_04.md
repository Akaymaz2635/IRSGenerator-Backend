- `IRSGenerator.API.csproj` → added `<Content>` item to copy `DispositionTemplates/**` to output directory

**New Service: `NcmSheetGenerator`**
- Opens `.docx` template as ZIP, replaces placeholders in raw XML
- Placeholders: `[SERIAL NUMBER]`, `[OPER]`, `[C-OP]`, `[QTY]`, `[C.CODE]`, `[NONCONFROMANCE PLACE HOLDER]` × 21 (7 rows × 3 per row)
- Auto-splits: if >21 NCs → second sheet as `TEMPLATENAME_2.docx`
- Output path: `C:\Users\{user}\Documents\Dispositions\{PROJECT}\{PART}\{SERIAL}\{OPERATION}\{TEMPLATE}.docx`

**Templates copied to project:**
